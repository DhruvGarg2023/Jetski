# Jetski Backend

The high-performance, real-time AI code reviewer backend for Jetski.

## Overview
Jetski is an AI-powered code review assistant that connects to your GitHub repositories, analyzes pull requests and commits using Google Gemini 2.5 Flash, and provides real-time feedback via Socket.IO.
